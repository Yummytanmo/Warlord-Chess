# Quickstart: Verify Modern Board

1.  **Start Dev Server**:
    ```bash
    npm run dev
    ```
2.  **Open Game**:
    - Navigate to `http://localhost:3000`.
    - Select a hero to start the game.
3.  **Visual Check**:
    - Board should render with Konva (Canvas).
    - Grid lines should be crisp.
    - Pieces should be circles with text.
    - Resize window -> Board should resize smoothly.
4.  **Interaction Check**:
    - Click a piece -> It should glow/highlight.
    - Valid moves -> Green dots appear.
    - Click valid move -> Piece moves.
