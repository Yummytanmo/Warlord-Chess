# Quickstart

## Testing Suicide Prevention
1. Start a game.
2. Maneuver pieces so that moving a piece would leave your King exposed to an enemy piece.
3. Attempt that move.
4. Verify that the move is rejected and a "General!" (or similar) prompt appears.

## Testing Restart/Reselect
1. Start a multiplayer game.
2. Click "Restart Game" in the Game Status panel.
3. Verify opponent receives a dialog.
4. If accepted, verify board resets.
5. Click "Reselect Hero".
6. Verify opponent receives a dialog.
7. If accepted, verify game returns to Hero Selection screen.
